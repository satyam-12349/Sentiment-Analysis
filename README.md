# Sentiment Analysis using Machine Learning

A Machine Learning based Sentiment Analysis web application built using **Python**, **Scikit-learn**, and **Streamlit**. This application analyzes text input and predicts whether the sentiment is **Positive** or **Negative**.

## 🚀 Features

* User-friendly Streamlit web interface
* Predicts sentiment from text input
* Machine Learning model trained using Scikit-learn
* Text vectorization using NLP techniques
* Fast sentiment prediction
* Easy deployment on Streamlit Cloud

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* NLTK
* Joblib

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/satyam-12349/Sentiment-Analysis.git
```

Go to the project directory:

```bash
cd Sentiment-Analysis
```

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run untitled.py
```

## 🧠 How It Works

1. User enters a text message.
2. The text is processed using NLP techniques.
3. The trained Machine Learning model analyzes the text.
4. The application displays the predicted sentiment:

   * Positive 😊
   * Negative 😞
